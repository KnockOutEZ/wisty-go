## Requirements
- Go 1.18 or above

## Running the project
Clone the repository
```bash
git clone git@github.com:KnockOutEZ/wisty-go.git && cd wisty-go
```
Install the dependencies
```bash
go get
```
Build the project
```bash
go build
```
Install the project
```bash
go install
```
Run the project from anywhere now
```bash
wisty-go --help
```
